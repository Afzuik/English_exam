**主编：李芝 | 中国人民大学出版社 (2025.7) | 共7个单元**

---

## 教材概述

本教材的编写宗旨是帮助学生系统掌握英语学术论文的篇章结构、语言特点、写作要求和学术规范。编写思路侧重于分析学术论文各部分的语步（move）和文体特征，引导学生把握英语学术论文的语篇特点和行文规范。全书共分七个单元：

- **Unit 1**：写作基础（Writing Foundation）—— 基本策略、语言正式性、信息表达逻辑性
- **Unit 2-7**：分别对引言、材料与方法、结果、讨论与结论、摘要、标题进行深入分析

## Unit One — Writing Foundation（写作基础）

**定位**：为学术写作打基础，介绍核心策略和语言技巧。

### 1. 学术写作策略

- **理解读者（Understand Your Audience）**：明确目标受众（导师、学位委员会成员、期刊评审人、会议学者），读者的知识水平决定了论文的内容深度和写作风格
- **写作策略清单**：
  - 以高水平的经典期刊论文为模版，关注其组织结构和用语
  - 每日坚持写作，一个月内完成初稿
  - 培养读者意识，清楚什么需要写、什么不需要写
  - 主动寻求导师和学长的指导
  - ⚠️ 避免：先写中文再翻译、坐等导师告诉写什么

### 2. 逻辑（Logic）

- **问题-解决模式（Problem-Solution Pattern）**：
  - **Situation** — 背景信息
  - **Problem** — 当前研究的原因/前人研究的局限
  - **Solution** — 解决方法
  - **Evaluation** — 评估解决方案的意义
- **逻辑连接词（Logical Connectors）分类**：

  | 类别 | 常用词汇 |
  |------|---------|
  | Addition | furthermore, in addition, moreover |
  | Adversativity | although, even though, despite, however, nevertheless |
  | Cause & Effect | because, since, therefore, as a result, consequently, hence, thus, due to |
  | Clarification | in other words, that is, i.e. |
  | Contrast | while, whereas, in contrast, however, on the other hand, conversely, unlike |
  | Illustration | for example, for instance |
  | Intensification | on the contrary, as a matter of fact, in fact |

- 通过对比学术范文与学生习作，识别主题句缺失、逻辑谬误（如错误归因）等常见问题

### 3. 风格（Style）

- **Swales & Feak（1994）十条学术写作原则**：
  1. 用 "we" 而非 "I"（现代学术写作中已更灵活）
  2. 避免缩写（如 don't → do not）
  3. 避免否定形式（如 not many → few）
  4. 避免模糊表达（如 a lot of → many / a great deal of）
  5. 不直接称呼读者（避免 you）
  6. 使用间接疑问句
  7. 副词置于句子中间位置
  8. 不拆分不定式（to boldly go → to go boldly）
  9. 避免冗余表达
  10. 灵活使用主动和被动语态

### 4. 语言聚焦

- **正式性（Formality）**：
  - 标准语法、完整句子、精确词汇
  - 避免俚语、口语化表达和缩写
  - 例句转换：cut down → reduce / gone up → increased / a bit → somewhat

- **信息流（Information Flow）—— 旧信息 → 新信息**：
  - 句子从左（已知信息/主题）向右（新信息/评论）推进
  - 四种指示旧信息的方式：重复关键词 / 使用同义词 / 代词或指示代词 / This/These + 总结名词

- **This + 总结名词**：
  - 替代重复的指代手段
  - 如：this phenomenon / this approach / this issue / this advantage / these perspectives

- **名词化（Nominalization）**：
  - 将动词或形容词转化为名词，使表达更正式、更抽象
  - 例句：We analyzed the data → An analysis of the data was conducted
  - 动词-名词对照：achieve → achievement / approach → approach / assess → assessment / assume → assumption / create → creation

## Unit Two — Introduction（引言）

**定位**：论文的"门面"和"地图"，从宽泛到具体的漏斗形结构。

### 1. 信息惯例

- **漏斗形结构（Funnel Structure）**：
  - 从一般背景信息 → 逐渐聚焦 → 具体研究问题
  - 开篇宽泛 → 逐步收窄 → 结尾明确指出研究目的

- **CARS 模型**（Creating a Research Space，Swales & Feak 2012）：

  | 语步 | 功能 | 可选/必选 |
  |------|------|----------|
  | **Move 1** | 确立研究领域 | |
  | 1a | 展示重要性 | 可选 |
  | 1b | 综述前人研究 | 必选 |
  | **Move 2** | 建立研究空位 | |
  | 2a | 指出空白 | 必选 |
  | 2b | 延伸前人知识 | 可选 |
  | **Move 3** | 占据空位 | |
  | 3a | 概述研究目的 | 必选 |
  | 3b | 列举研究问题/假设 | 可选 |
  | 3c | 宣布主要发现 | 可选 |
  | 3d | 陈述研究价值 | 可选 |
  | 3e | 说明论文结构 | 可选 |

### 2. 定义写作（Writing Definitions）

- **短定义（Short Definition）**：使用标志词（i.e., known as, defined as, called），通常出现在括号或逗号中
- **句子定义（Sentence Definition）**：
  - 标准公式：**术语 + be + 类别词 + which/that 从句 + 区分细节**
  - 例：A solar cell is a device that converts solar energy into electricity.
- **⚠️ 循环定义（Circular Definition）**：定义中包含被定义词本身（如"自由就是自由的状态"），是学术写作中必须避免的问题
- **扩展定义（Extended Definition）**：从一般到具体，通过以下方式延伸：
  - 分类（Classification）
  - 历史背景（Historical background）
  - 示例（Examples）
  - 组成部分（Components）
  - 应用（Applications）

### 3. 引用（Citation）与转述（Paraphrasing）

- **两种引用模式**：
  - **信息突出引用**（Information-prominent）：侧重内容/研究发现，括号内含年份
  - **作者突出引用**（Author-prominent）：侧重作者身份，如 "Harper (2008) found that..."

- **引用中的动词时态**：
  | 引用类型 | 时态 | 示例 |
  |---------|------|------|
  | 单项研究引用 | 一般过去时 | Rutherford (1990) found... |
  | 研究领域引用 | 现在完成时 | ...has been observed in a large number of studies |
  | 当前知识状态 | 一般现在时 | Plasticity in plants is... |

- **转述策略**：
  - 替换同义词
  - 改变词序
  - 调整句子结构

### 4. 建立研究空位（Establishing a Niche）与占据空位（Occupying the Niche）

- **建立空位的四种方法**：
  1. 反驳（Counter-claiming）：某观点是错误的
  2. 指出空白（Indicating a gap）：缺少某些内容
  3. 提出疑问（Raising a question）：某些内容不明确
  4. 延续传统（Continuing a tradition）：需要补充更多内容

- **目的陈述（Purpose）vs 性质陈述（Nature）的区分**
- **目的陈述的时态**：
  - 主语为文本类型（paper, article）：一般现在时 — This paper proposes...
  - 主语为调查类型（experiment, study）：一般过去时 — The goal was to...
- **研究问题**：直接问句 → 间接问句转换
- **情态动词表达研究价值**：may > can > could > would（从强到弱）

### 5. 语言聚焦

- 旧信息→新信息的推进模式（同 Unit 1）
- 四种指示旧信息的方式：重复关键词 / 同义词 / 代词 / This/These + 总结名词

## Unit Three — Materials and Methods（材料与方法）

**定位**：确保研究的**透明性（Transparency）**和**可重复性（Replicability）**。

### 1. 信息惯例

- **10个常见语步要素（Move Elements）**：
  1. 概述（Overview / method summary）
  2. 研究目的/问题/假设
  3. 研究地点（Study site）
  4. 参与者/材料（Participants / Materials）
  5. 测量或抽样技术（Measurement / Sampling techniques）
  6. 实验程序（Experimental procedures）
  7. 限制条件（Limitations）
  8. 变量（Variables）
  9. 数据分析（Data analysis）
  10. 伦理考量（Ethical considerations）

> ⚠️ 语步顺序不固定，可能循环出现

### 2. 材料信息要素

- **材料类别**：
  - 研究地点（Study sites）
  - 实验室设备（Laboratory equipment）
  - 野外设备（Field equipment）
  - 人类/动物对象（Human/Animal subjects）
  - 天然物质（Natural substances）
  - 人工材料（Artificial materials）
  - 调查问卷（Questionnaires）
  - 计算机模型（Computer models）
  - 数学模型（Mathematical models）

- **定量细节要求**：
  - 来源（Source）
  - 制备方法（Preparation methods）
  - 技术规格（Technical specifications）
  - 选用理由（Rationale for selection）
  - 研究地点应包含经纬度（GPS coordinates）
  - 化学品应注明制造商信息（Manufacturer details）
  - 自研材料需详细说明制备过程

### 3. 语言聚焦

- **信息流**：过程描述中的旧信息→新信息模式至关重要
- **程序描述**：逐步、按时间顺序描述
- **-ing 从句表示因果**：
  - 使用 "thus/thereby + -ing 从句" 替代传统连接词
  - 结构：主句 + (thus/thereby) + -ing从句
  - 可用于：描述过程结果、问题结果、解决方案结果

- **动词时态**：
  | 场景 | 时态 |
  |------|------|
  | 描述所选样本 | 一般过去时 |
  | 描述特制材料 | 一般过去时 |
  | 描述实验程序步骤 | 一般过去时 |
  | 描述一般人群 | 一般现在时 |
  | 描述常规材料/设备 | 一般现在时 |

- **动词语态**：
  - **被动语态**：传统上用于描述程序，去除施动者，增强客观性
  - **主动语态**：当研究者很重要时使用；描述自然过程时使用；部分期刊偏好主动语态
  - "by + 人" 结构在正式写作中罕见；"by + 过程/非人施动者" 更常用
  - **核心原则**：根据信息流选择语态

## Unit Four — Results（结果）

**定位**：客观报告主要发现，提供简短的有针对性的数据评论。

### 1. 信息惯例

- **结果部分的七大功能**（Swales & Feak 2012）：
  1. 突出结果（Highlight results）
  2. 用数据支持论点（Support arguments with data）
  3. 评估理论（Evaluate theory）
  4. 比较不同数据集（Compare different sets of data）
  5. 评估数据可靠性（Evaluate data reliability）
  6. 讨论数据含义（Discuss implications）
  7. 提出建议（Make recommendations）

- **数据评述三要素**：

  | 要素 | 英文 | 功能 | 示例 |
  |------|------|------|------|
  | 要素1 | Location element | 指出结果所在图表 | Table 2 shows... / As can be seen in Fig. 1... |
  | 要素2 | Highlight element | 呈现最重要的发现 | The most notable finding is... |
  | 要素3 | Comment element | 对结果进行评论/解释 | This suggests that... / This finding indicates... |

- **两种组织模式**：
  - **交替模式（Alternating Pattern）**：Result 1 + Comment 1 → Result 2 + Comment 2 → ...
  - **序列模式（Sequential Pattern）**：All Results → All Comments

### 2. 可视化信息

- **表格（Table）vs 图（Figure）的选择原则**：

  | 表格适用于 | 图适用于 |
  |-----------|---------|
  | 展示大量数据 | 展示数据关系 |
  | 显示精确数值 | 部分与整体关系 |
  | 方便具体比较 | 流程顺序 |
  | — | 分布或趋势 |

- **图表标注规范**：
  - 编号：Table 1, Table 2... / Figure 1, Figure 2...（简称 Fig. 1）
  - 表标题在表上方，图标题在图下方
  - 描述性标题和说明文字

- **趋势描述词汇**：

  | 趋势方向 | 常用词汇 |
  |---------|---------|
  | 上升 | increase, rise, grow, climb, surge, soar |
  | 下降 | decrease, fall, drop, decline, plummet, plunge |
  | 稳定 | remain steady, stabilize, level off, plateau |
  | 峰值 | peak, culminate in, reach a maximum |
  | 谷底 | trough, bottom out, reach a minimum |
  | 波动 | fluctuate, oscillate, vary |

### 3. 语言聚焦

- **主张强度（Strength of Claims）**：
  - 学术写作中的核心立场："有把握的不确定性"（confident uncertainty）
  - 使用模糊限制语（hedging）软化过于肯定的主张
  - 常见手段：情态动词（may, might, could）、试探性动词（suggest, indicate, appear）

- **指示性陈述 vs 信息性陈述**：
  - **指示性陈述（Indicative）**：仅指出数据存在但不详述 — "Table 2 shows the results..."
  - **信息性陈述（Informative）**：提供具体细节和解释 — "As shown in Table 2, the majority of participants..."

- **As-从句**："As shown in... / As illustrated by..." 等结构的特殊用法

- **时态使用**：
  | 功能 | 时态 |
  |------|------|
  | 定位要素（指向图表） | 一般现在时 |
  | 报告发现 | 一般过去时 |
  | 评论发现 | 一般现在时（常带情态动词和试探性动词） |

- **意外发现**：使用 Interestingly / Unexpectedly / Surprisingly / Contrary to our initial hypothesis...

## Unit Five — Discussion and Conclusions（讨论与结论）

**定位**：从具体到一般，解释发现、与现有文献比较、承认局限性、展望未来。

### 1. 信息惯例

- **论文末尾的常见标题模式**：
  - "Results and Discussion"（结果与讨论合并）
  - "Results" + "Discussion"（分别呈现）
  - "Results" + "Conclusions"
  - "Results" + "Discussion" + "Conclusions"

- **五语步结构（Five-Move Structure）**：

  | 语步 | 内容 |
  |------|------|
  | **Move 1** | 背景信息（Background information） |
  | **Move 2** | 总结和报告关键结果（Summarize & report key findings） |
  | **Move 3** | 评论关键结果 —— 解释、比较、评价（Comment on key results） |
  | **Move 4** | 陈述研究局限性（State limitations） |
  | **Move 5** | 对未来实施/研究提出建议（Recommendations for future research） |

### 2. 局限性处理（Handling Limitations）

- **局限性的四个目的**：
  1. 透明性与诚信（Transparency & integrity）
  2. 将发现置于语境中（Contextualize findings）
  3. 提供反思机会（Opportunity for reflection）
  4. 为未来研究奠定基础（Lay foundation for future research）

- **两种表达方式**：
  - **A 类**：被动限制句（Passive limiting sentences）
  - **B 类**：主动承认句（Active acknowledging sentences）

- **减轻局限性负面影响的五种策略**：
  1. 将局限归因于当前理论的不完善
  2. 对局限进行清晰解释
  3. 说明局限带来的后果
  4. 提出改进局限的建议
  5. 使用替代词汇（如用 "challenge" 替代 "weakness"）

### 3. 语言聚焦

- **背景信息**：讨论开头可以重述以下内容之一：
  - 研究目的（Research purpose）
  - 假设/预测（Hypothesis/Prediction）
  - 研究方法（Methods）
  - 文献回顾（Literature review）
  - 最重要的发现（Most important findings）

- **与引言的呼应**：讨论的开头应与引言末尾的研究目的/问题形成对应关系

- **原因解释**：提供可能的原因和机制来解释研究发现 —— 这是讨论部分的核心

- **模糊限制语（Hedging）**：减轻过度断言的风险
  - 示例表达：It is possible that... / This may be due to... / The findings suggest that...

- **时态使用**：
  | 信息要素 | 时态 |
  |---------|------|
  | 目的、假设、发现重述 | 一般过去时 |
  | 解释、局限性 | 过去时/现在时/情态动词 |
  | 与现有研究比较 | 一般现在时 |
  | 含义、建议 | 现在时/情态动词/试探性动词 |

- **讨论中的引用**：支撑关键论点 / 处理局限性 / 扩展主题 / 讨论含义 / 致谢前人

## Unit Six — Abstract（摘要）

**定位**：论文的高度浓缩版本，出现在文章开头，帮助读者快速判断论文相关性。

### 1. 五语步模型（B-P-M-R-C）

| 语步 | 英文 | 中文 | 回答的问题 |
|------|------|------|-----------|
| **B** | Background | 背景/引言/情境 | 我们对该主题了解多少？为什么重要？ |
| **P** | Purpose | 目的/当前研究 | 该研究是关于什么的？ |
| **M** | Methods | 方法/材料/受试者/程序 | 如何（在哪里）进行的？ |
| **R** | Results | 结果/发现 | 发现了什么？ |
| **C** | Conclusion | 结论/讨论/含义/建议 | 发现意味着什么？ |

### 2. 信息惯例

- **开头句的四种类型**（Swales & Feak 2009）：
  1. **类型1**：以背景信息开头
  2. **类型2**：以问题或不确定性开头
  3. **类型3**：以目的或目标开头
  4. **类型4**：以当前研究行动开头

- **方法语步的压缩**：由于摘要字数限制，方法描述需要精简（通常建议 < 40词）
- **结果语步**：呈现最重要的发现，由一般到具体；定量研究应提供精确数字
- **结论语步**：强调研究意义，可以以一般性含义或具体建议结尾

### 3. 语言聚焦

- **动词时态与语步的对应**：

  | 语步 | 常用时态 |
  |------|---------|
  | Background（背景） | 一般现在时 |
  | Purpose（目的） | 一般现在时（文本类型主语）/ 一般过去时（调查类型主语） |
  | Methods（方法） | 一般过去时 |
  | Results（结果） | 一般过去时 |
  | Conclusion（结论） | 一般现在时 / 情态动词 |

- **That-从句在结果语步中的强调作用**：
  - 名词短语：The results showed **an increase in efficiency**
  - That-从句（更有强调效果）：The results showed **that efficiency increased significantly**
  - That-从句更常跟在无灵主语（The findings... / The results...）后面

## Unit Seven — Title（标题）

**定位**："摘要的摘要"——用最少字数传达最核心信息。三个核心原则：**简洁、清晰、信息丰富**。

### 1. 信息惯例

- **三种语法结构模式**：

  | 类型 | 结构 | 示例 | 使用频率 |
  |------|------|------|---------|
  | 名词短语标题 | 以名词性短语为核心 | "Impact of different eddy covariance sensors on..." | 最常见 |
  | 复合标题 | 两部分或以上用标点连接 | "Urban forest governance: Towards a framework..." | 较常见 |
  | 完整句子标题 | 以完整句子形式出现 | "Biophysical and lightning characteristics drive lightning-induced fire occurrence..." | 医学/生物等领域常见 |

- **三种大小写模式**：
  1. **Sentence case**（句首字母大写）：仅首字母和专有名词大写
  2. **Title case**（首字母大写）：每个实词的首字母大写
  3. **All caps**（全大写）：全部大写（较少使用）

- 标题中斜体用于物种学名（如 *E. coli*）

### 2. 语言聚焦

- **信息丰富性（Informativeness）**：标题应包含关键要素，指示论文主要内容
  - ✅ 更好："Effects of nitrogen deposition on soil microbial diversity in subtropical forests"
  - ❌ 较差："A study on soil microbes"

- **简洁性（Conciseness）**：避免冗余表达
  - 删除 "A study of...", "Investigation on...", "Observation on...", "Research on..." 等冗余词
  - ✅ 简洁："Factors affecting consumer trust in e-commerce"
  - ❌ 冗长："A comprehensive study and investigation of various factors that affect..."

- **清晰性（Clarity）**：动词和介词的恰当使用有助于读者理解标题各要素之间的关系

- **标题翻译四步法**：
  1. 确定中心名词（Identify the head noun）
  2. 划分意群（Break into semantic groups）
  3. 按逻辑关系排列（Arrange by logical relationships）
  4. 用介词连接（Connect with appropriate prepositions）
